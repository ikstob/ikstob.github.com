About

“botski” is quite simply a selection of tools to automate some of the more tedious tasks related to Internet Marketing. These tools include being able to harvest thousands of links from the major search engines, fetch and scrape web pages for particular patterns, check for page and domain related attributes (PageRank) and some data management tasks.

At the moment these tools are all command-line based and written entirely in Java, they work on Windows, Mac and Linux and are ideal for running remotely on cheap Linux VPS’s. Some web/desktop user interfaces may be added in the future but all the tools will run happily on an inexpensive 256MB VPS.

Getting Started

First things first, you need to download the software - you have two choices:
  1. Download the ready-to-go compiled code (ZIP or TAR)
    2. or,
  2. Build it yourself from source code (advanced users only)

Download the ready-to-go compiled code

Choose the version you’d like the download in (ZIP or TAR) and save it on your computer. Extract the archive into a new “botski” directory, this is where you will run the program from.

Build it yourself from source code (advanced users only)

Assuming you have Java JDK 1.6+ and Apache Maven installed.

$ git clone git://github.com/ikstob/botski.git
$ cd botski
$ mvn package

Basic Usage

The following is an example of Harvesting links from the major search engines using a list of footprints in the given files. These are in an identical format to other tools like Scrapebox etc.

$ botski Harvest path/to/footprints1.txt path/footprints2.txt
